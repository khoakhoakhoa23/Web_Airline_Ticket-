# Thư mục hình ảnh Public

## Cách sử dụng

Đặt các file hình ảnh vào thư mục này và chúng sẽ được phục vụ trực tiếp từ root URL.

## Hình ảnh cần thiết

### airplane-sky.jpg
- **Mô tả**: Hình máy bay trắng bay trên bầu trời xanh với mây
- **Đường dẫn**: `/images/airplane-sky.jpg`
- **Sử dụng tại**: Home, Login, Register pages

## Lưu ý

- File trong thư mục `public/` sẽ được copy trực tiếp vào build
- Đường dẫn bắt đầu từ `/` (root)
- Không cần import, chỉ cần sử dụng đường dẫn tuyệt đối
